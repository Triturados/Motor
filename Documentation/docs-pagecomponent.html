Transform

Clase que contiene el componente Transform de los objetos.

#include <Transform.h>

Ejemplo de uso del componente en Lua:

local trcam = cam:addComponent('Transform')
		trcam:sendMsg([[
		scale: 2,2,2;
		position: 0,40,80;
		rotation: 0,0,0;
	]])

Nota

En el codigo mostrado se entiende que existe el objeto cam

Timer

Clase que contiene el componente Timer de los objetos.

#include <Timer.h>

Funciones publicas disponibles desde este componente:

static Timer* invoke(action funct, float time) Llama a la funcion funct al pasar time segundos, solo 1 vez.
static Timer* repeat(action funct, float interval) Llama a la funcion funct al pasar los segundos de la variable interval.
static Timer* deleteGameObject(GameObject* gameObject, float time) Borra el objeto gameObject al pasar time segundos.
static Timer* deleteComponent(Component* component, float time) Borra el componente component al pasar time segundos.
void noloop() Hace que el temporizador deje de funcionar en bucle.
void loop() Hace que el temporizador pase a funcionar en bucle.
void resume() Permite al temporizador volver a funcionar.
void pause() Pausa el temporizador.
void restart() Reinicia el temporizador.
void stop() Para el temporizador.
void setValue(action f, float t) Cambia la funcion que se ejecuta al acabar el temporizador por f y el tiempo que tarda el temporizador en ejecutar la orden con la variable t

Text

Clase que contiene el componente Text de los objetos.

#include <Text.h>

Ejemplo de uso del componente en Lua:

local textPruebaText = textPrueba:addComponent("Text")
	textPruebaText:sendMsg([[
		position: 50, 50, 0;
		fontName: chikory
		mainColor: 0.3, 1.0, 0.6, 1.0;
		bottomColor: 0.5, 1.0, 0.2, 1.0;
		topColor: 0.5, 0.0, 0.5, 1.0;
		textScale: 0.05
		alignment : 0
		text: textodeejemplo
	]])

Nota

En el codigo mostrado se entiende que existe el objeto textPrueba.

Valores que se le pueden enviar por Lua:

position Vector 3 que coloca el texto en pantalla.
fontName Nombre de la letra que va a usar el componente.
mainColor Color principal del texto en vector 4 en el formato rgb siendo el 4 componente la opacidad.
bottomColor Color por debajo del texto.
topColor Color superior del texto.
textScale Tamaño que va a tener el texto por pantalla.
alignment Alineado del texto (Izquierda = 0, Centrado = 1, Derecha = 2).
text String que corresponde al texto que saldra por pantalla.

Funciones publicas disponibles desde este componente:

void receiveMessage(Utilities::StringFormatter& info)) Recibe la información enviada por Lua.
void setVisibility(bool mode = true) Cambia la visibilidad del texto en funcion del boleano mode.
void changeText(std::string mssg) El texto que se pasa a ver por pantalla es el enviado en nel string mssg.

Sound

Clase que contiene el componente Sound de los objetos.

#include <Sound.h>

Ejemplo de uso del componente en Lua:

player:addComponent("Sound"):sendMsg([[
		soundName: ejemplo.mp3; 
		channel: music; 
		loop: true
		]])

Nota

En el codigo mostrado se entiende que existe el objeto player.

Valores que se le pueden enviar por Lua:

soundName Nombre del archivo de musica que se quiere reproducir con su extensión.
channel Canal en el que se reproducira el sonido, es un enum con 4 valores effects, environment, voices y music.
loop Booleano para ejecutar el sonido en bucle o no.

Funciones publicas disponibles desde este componente:

void receiveMessage(Utilities::StringFormatter& info)) Recibe la información enviada por Lua.
playSound() Ejecuta el sonido.
void releaseSound() Borra de memoria el sonido.
void setSpeed(float s) Seleccion a la velocidad a la que se quiere reproducir un sonido, siendo 1 el valor predeterminado del sonido.
void pause(bool pause) Vuelve a reproducir el sonido por donde lo habvia dejado o lo para en funcion del boleano pause.
void setVolume(float volume) Selecciona el volumen del sonido a reproducir a traves de el float volume.
